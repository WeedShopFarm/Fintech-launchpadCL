🚀 QUICK START (5 MINUTES)
════════════════════════════════════════════════════════════════════════════

  1. Navigate to backend:
     $ cd fintech-backend

  2. Install dependencies:
     $ npm install

  3. Setup database:
     $ npm run migration:run

  4. Start development server:
     $ npm run start:dev

  5. Server will run on http://localhost:3000
